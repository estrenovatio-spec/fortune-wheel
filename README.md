# Колесо фортуны — SG Capital

Отдельный лендинг с колесом призов для финансового советника Алексея Шаргатова.

- Next.js 14, Tailwind
- Уведомления в Telegram при заборе приза
- Опционально Google Sheets (`type: "wheel"` в webhook)

## Быстрый старт

```bash
npm install
cp .env.example .env.local
npm run dev
```

Деплой: см. [DEPLOY.md](./DEPLOY.md).
